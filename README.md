# A-CLI
Un bash script para ver animes sub español desde tu terminal usando mpv.

## Dependencias
1. mpv
2. curl
3. grep
4. jq

# Instalacion
Clone el repositorio git `https://github.com/IamJony/A-CLI.git` y conceda permisos de ejecucion a los scripts `chmod +x A-CLI.sh ./modules/*`
